API Documentation

Group 1: User APIs

GET /api/users

Fetches a list of users.

Parameters:

  • query: Optional. Search query to filter users.

Response:

{
    "users": [
        {
            "id": 1,
            "name": "John Doe",
            "email": "john.doe@example.com"
        },
        ...
    ]
}

Group 2: File APIs

POST /api/files/upload

Uploads a new file.

Body:

  • file: Required. The file to be uploaded.

Response:

{
    "success": true,
    "fileId": 123
}
